# Conversor de Temperaturas

Programa en JavaScript que transforma temperaturas de grados Celsius a Farenheit y viceversa, con dos decimales de precisión.

## Uso

Escribir la temperatura en el cuadro de texto del siguiente modo:

Número entero o decimal + letra (F o C, mayúscula o minúscula).

Se pueden poner los espacios que se quiera entre el número y la letra.
